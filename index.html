Mis Proyectos

Api Rest dedicada en Java

Java MySQL

Esta API Rest ha sido desarrollada completamente en **Java**, utilizando **Spring Boot** para facilitar la creación de servicios web y garantizar una arquitectura modular y escalable. Para la gestión de la base de datos, se ha implementado Hibernate como ORM (Object-Relational Mapping), lo que permite la manipulación eficiente de los datos almacenados en MySQL. Esta API está diseñada para manejar información de usuarios, incluyendo autenticación segura y gestión de credenciales mediante técnicas de cifrado. Además, la API ofrece un CRUD completo (Create, Read, Update, Delete) para administrar usuarios y sus permisos. Se han implementado controladores RESTful siguiendo buenas prácticas de arquitectura, junto con una capa de servicios bien definida para separar la lógica de negocio del acceso a datos.



Proyecto Academia

Python Odoo

El proyecto "Academia" es un sistema de gestión de actividades y participantes desarrollado en Odoo utilizando Python como lenguaje principal. Se compone de tres modelos principales: Activities, además de una relación Many-to-One con Participants, mostrando el progreso de inscripciones con una barra de progreso; Participants, que almacena datos de los inscritos incluyendo nombre, email, fecha de nacimiento, fecha de inscripción, actividad relacionada y un responsable vinculado a res.users, permitiendo también asociar una imagen; y Booking, que establece una relación Many-to-Many con Activities para organizar todas las actividades en un solo módulo. Gracias a la flexibilidad de Odoo y la potencia de Python, este sistema permite una gestión eficiente y modular de las actividades dentro de la academia.
