Mis Proyectos

Api Rest dedicada en Java

Java MySQL

Esta API Rest ha sido desarrollada completamente en **Java**. Para la gestión de la base de datos, se ha implementado Hibernate como ORM , lo que permite la manipulación eficiente de los datos almacenados en MySQL. Esta API está diseñada para manejar información de usuarios, incluyendo autenticación segura y gestión de credenciales mediante técnicas de cifrado. Además, la API ofrece un CRUD completo (Create, Read, Update, Delete) para administrar usuarios y sus permisos. Se han implementado controladores RESTful siguiendo buenas prácticas de arquitectura, junto con una capa de servicios bien definida para separar la lógica de negocio del acceso a datos.



Proyecto Academia

Python Odoo

El proyecto "Academia" es un sistema de gestión de actividades y participantes desarrollado en Odoo utilizando Python como lenguaje principal. Se compone de tres modelos principales: Activities, además de una relación Many-to-One con Participants, mostrando el progreso de inscripciones con una barra de progreso; Participants, que almacena datos de los inscritos incluyendo nombre, email, fecha de nacimiento, fecha de inscripción, actividad relacionada y un responsable vinculado a res.users, permitiendo también asociar una imagen. Gracias a la flexibilidad de Odoo y la potencia de Python, este sistema permite una gestión eficiente y modular de las actividades dentro de la academia.



Club Deportivo

Javascript Mysql Php Html Css

El proyecto "Club Deportivo" es una aplicación web desarrollada con PHP, JavaScript y MySQL, integrando APIs externas y dedicadas, además de gestionar sesiones de usuario. Su objetivo es administrar un gimnasio, permitiendo la gestión de socios, reservas, noticias, reseñas y otros servicios. Ofrece una experiencia dinámica y fácil de usar, combinando bases de datos eficientes con interactividad mediante JavaScript y comunicación con servicios externos a través de APIs.
